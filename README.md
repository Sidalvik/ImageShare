# Image Share

Веб-сайт для загрузки и просмотра изображений и видео.

## Функционал

- Загрузка изображений и видео через веб-форму
- Автоматическая организация файлов по датам (YYYY-MM-DD)
- Просмотр файлов в виде карточек, отсортированных по датам
- Копирование URL файлов в буфер обмена
- Просмотр файлов в полноэкранном режиме

## Требования

- PHP 7.0 или выше
- Веб-сервер (Apache/Nginx)
- Права на запись в директорию `uploads/`

## Установка

1. Загрузите файлы на сервер
2. Убедитесь, что директория `uploads/` существует и имеет права на запись (chmod 755)
3. Откройте сайт в браузере

## Структура файлов

- `index.php` - главная страница с формой и списком файлов
- `upload.php` - обработчик загрузки файлов
- `style.css` - стили
- `script.js` - JavaScript для интерактивности
- `uploads/` - директория для хранения загруженных файлов

## Поддерживаемые форматы

**Изображения:** JPG, JPEG, PNG, GIF, WEBP, BMP, SVG
**Видео:** MP4, WebM, OGG, MOV, AVI

Максимальный размер файла: 100MB
